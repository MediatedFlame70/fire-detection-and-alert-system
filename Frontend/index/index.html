How It Works

A 4-stage pipeline from drone capture to detection output, running in real time.

🚁

Drone Captures

Drone captures a 448×448 RGB image of the target area from high altitude.

🧠

CNN Extraction

3 CNN stages (MobileNetV3-style) extract local texture and color features.

🔍

Vision Transformer

2 ViT stages analyze global context and long-range spatial relationships.

🚨

Alert & Output

Model outputs Fire / Smoke / Neutral class + bounding box + confidence score.

Dataset

Trained on the Kaggle Fire and Smoke Dataset with structured annotations.

📦 Source

Kaggle Fire and Smoke Dataset by DataCluster Labs. Contains annotated fire and smoke images across diverse environments.

dataset_root/
├── images/

🏷️ Annotation Format

Each image is annotated with a class label and a bounding box in normalized [x, y, w, h] format.

{
  "filename": "image_001.jpg",
  "class": "fire",
  "bbox": [x, y, w, h]
}
